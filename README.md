## Escenario Elegido
Escenario B: Analisis de ventas

## Descripcion
El archivo `datos/ventas_empresa.csv` contiene el historial de transacciones de la empresa, incluyendo los siguientes campos:
- **id**: Identificador único de la transacción.
- **producto**: Nombre del artículo vendido.
- **cantidad**: Cantidad vendida.
- **precio**: Precio unitario.
- **fecha**: Fecha en la que se realizó la operación.

## Instrucciones de Ejecucion
1. Estar en el directorio raiz del repositorio.
2. Ejecutar el script principal utilizando Python:
   `python scripts/analisis_ventas.py`
3. Los resultados  se guardarán automáticamente en la carpeta `/resultados` como `grafico_ventas.png`.
